# AI-Based Employee Stress Detection and Analysis System

This project is a multi-modal AI system designed to detect and analyze employee stress levels using facial expressions, voice tone, and typing speed.

The system uses computer vision techniques for real-time facial emotion detection, audio signal processing for voice stress analysis, and typing speed evaluation to assess cognitive stress. These inputs are combined to generate an overall stress level (Low, Moderate, High).

The application also stores results in a SQLite database and includes role-based access for secure data viewing.

## Technologies Used
- Python
- OpenCV
- Librosa
- NumPy
- Scikit-learn
- SQLite

## Features
- Real-time facial emotion detection
- Voice tone analysis using pitch and energy
- Typing speed-based stress evaluation
- Multi-factor stress classification
- Database storage and secure access
